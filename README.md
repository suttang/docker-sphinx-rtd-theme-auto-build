# Automation sphinx building tool

## Install

```
git clone https://github.com/suttang/docker-sphinx-rtd-theme-auto-build
cd docker-sphinx-rtd-theme-auto-build
npm install
```

## Usage

### Development build

Open browser and auto build, refresh

```
npm run build:dev
```

### Production build

Refresh all documents

```
npm run build:prod
```
